# Voting Dapp

Smart Contract Functionality:

- all of the votes and candidate registration happens on chain

---

1. Anyone can start an election with:

- registration period
- voting period
- ending time

2. allow anyone to sign up as a candidate during the registration period
3. allow anyone to vote once during the voting period
